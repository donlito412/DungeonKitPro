# Dungeon Kit Pro

**Procedural Dungeon Generator for Unity**

Create complete dungeons with rooms, corridors, and decorations instantly.

## Features

- **5 Dungeon Styles** - Stone, Crypt, Mine, Temple, Sewer
- **Procedural Generation** - Unique layouts every time
- **Room Variety** - Multiple sizes and layouts
- **Corridor System** - Connects all rooms automatically
- **Decorations** - Torches, pillars, treasure chests
- **Lighting** - Atmospheric point lights

## Installation

1. Import this package into your Unity project
2. Go to **Tools > Dungeon Kit Pro**

## Usage

1. Select your dungeon style
2. Adjust room count and sizes
3. Enable/disable features
4. Click "GENERATE DUNGEON"

## Requirements

- Unity 6.3 LTS or later
- Universal Render Pipeline (URP)
